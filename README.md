# For Teachers
### Disclaimer
Hey students, please read this first before going on.  By all means, you are not forbidden from reading beyond this point in the read-me file, but there are some things to consider before you do go on.  First of all, you may find content here that ends up being spoilers for some lessons your teacher might assign in the future.  If that does end up happening, you are NOT allowed to be a jerk and spoil the lesson for all the other students in the class.  It does not exclude you from doing the lesson again.  It is a challenge for you to find some other way you can grow by doing the same lesson a second time.

Also, the language and reading difficulty is going to spike considerably from here on out.  So remember to take things slowly, and give yourself time to digest and understand what is below.  It is intended for teachers, so that will mean more looking up terms you don't know and learning it in smaller chunks.  But the very drive and curiosity that sends you into this part of the readme is the same thing that drove the greatest inventors and thinkers in our history to make the world a better place.  So go read on, but be read to crash and burn, and pick yourself up again and rise like the phoenix you are.
